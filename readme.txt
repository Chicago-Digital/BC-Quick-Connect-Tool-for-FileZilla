
                    BC Quick Connect Tool
        Tool to Quickly Connect to a BC Site via SFTP
            Created by Greg Pace @ Chicago Digital

Pass in the following information and automatically connect using FileZilla on Windows.		

URL Prefix: "[URL Prefix].worldsecuresystems.com"
Email: "BC Email that has admin rights"
Password: "BC Password"

Runs via command line. See source code in source.bat.
